搭建环境

  • 下面配置好的环境可以备份，每次写代码时复制一份。或者自定义一个 Xcode 模板。这是笔者的代码仓,模板的 tag 是 OpenGL_template。模板下载后, 放在~/Library/Developer/Xcode/Templates/Project Templates下。如果编译报头文件错误，可能要在Build Settings中修改PublicOpenGLHeaderPath为$(SRCROOT)/你的工程名。然后如图使用

    git clone 克隆或下载一个仓库单个文件夹

  • 开始搭建环境。创建一个 mac cocoa app。删除不需要的文件。

  • 编译报缺少 xxx.entitlements 的错。

  • 添加 OpenGL.framework 和 GLUT.framework。

    在弹出的窗口中搜索上面两个库添加即可。

  • 创建 main.cpp 文件（选择 macos c++ file，不需要main.hpp）。

  • 将 include 文件夹拖入项目。这里面有着色器容器类等。记住勾选 copy item if needed。 include在这

  • 修改头文件搜索路径

  • 在 main.cpp 中添加代码，应该能够编译通过了。
